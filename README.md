# GesLeitura - VendingGuard SaaS

Sistema de gestão para máquinas de venda automática (vending machines).

## Funcionalidades

- Gestão de operações (seções, rotas, pontos, operadores)
- Registro de leituras
- Relatórios de vendas
- Gestão financeira (caixa geral, cotas de sócios)
- Administração de usuários
- Autenticação Firebase

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm run dev`
